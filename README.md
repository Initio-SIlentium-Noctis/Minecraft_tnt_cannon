# Tnt Cannon

## About

This mod is highly influenced by [Fox's Handheld Cannon](https://www.curseforge.com/minecraft/mc-mods/foxs-handheld-cannon) which adds a tnt cannon weapon. You can load it with different ammo types and shoot projectiles. Currently you can shoot TNT, cannon balls, fire charges or simply use powder for rocket jump. –ê few enchantments are available also. Mod supports English and Russian languages.

## Requirments

[Fabric API 0.76.0+](https://fabricmc.net/use/installer/)  

[Fabric loader 0.14.22+](https://fabricmc.net/use/installer/)  

Installation: Drop .jar file into mods folder.

## Enchantments
**Quick Reload** - reduces reload time  
**Fast Fuse** - reduces tnt fuse time  
**Sturdy Footing (1-3)** - reduces recoil  
**Volatile Gunpowder (1-3)** - increases shooting projectile range and overall recoil
